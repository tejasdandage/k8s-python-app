# 🚀 Python App on Kubernetes

## 📌 Project Overview
This project demonstrates deploying a Python (Django) application on Kubernetes using Docker and Minikube.

## 🛠️ Tech Stack
- Docker
- Kubernetes
- Minikube
- Python (Django)

## ⚙️ Steps
1. Build Docker image
2. Deploy using Kubernetes Deployment
3. Expose using NodePort Service
4. Access via Minikube

## 📸 Output
App accessible at:
http://<minikube-ip>:<nodeport>/demo/
